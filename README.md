# fastdfs_nginx_lua
==================
fastdfs nginx lua 自动生成缩略图，无group版

注意点：
----------------
* 修改 fastdfs.lua 中 fdfs:set_tracker("192.168.1.196", 22122)  为自己的IP
* nginx.conf 中添加 user root;
* 安装 GraphicsMagick
